# 🟠 대칭키 (Symmetric Key)
```
암호화에서 사용되는 키와, 복호화에서 사용되는 키가 동일한 암호키 (대칭키 암호화 알고리즘 이라고도 불린다.)
하나의 키를 양쪽(client & server) 가 같이 사용한다. 
```

## 🔸 대칭키 장점
공개키 암호화 방식에 비해서 암호화 및 복호화 속도가 빠르다.
